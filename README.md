# Инструкция по установке информационной системы "Pixel Deck"
## Требования
- XAMPP версии 3.3.0 или выше
- Веб-браузер (Google Chrome, Edge и т.д.)
## Настройка проекта
1) Скачать папку проекта и перенести в директорию C:\xampp\htdocs\
2) Запустить XAMPP
3) Включить Apache
4) Включить MySQL
5) Открыть phpMyAdmin (нажать кнопку Admin рядом с MySQL)
6) Скачать файл pixel_deck.sql
7) Создать базу данных с названием mebel и кодировкой utf8mb4_general_ci
8) Импортировать скачанный файл (вкладка "Импорт")
## Запуск проекта
1) Запустить XAMPP
2) Включить Apache
3) Включить MySQL
4) Открыть в браузере ссылку http://localhost/"Имя папки с проектом"/index.php
